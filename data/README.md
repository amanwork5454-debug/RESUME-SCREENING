# Dataset

The training dataset is **not committed to this repository** (it is listed in `.gitignore`).

## Download

1. Go to the Kaggle dataset page:
   **[Resume Dataset – by gauravduttakiit](https://www.kaggle.com/datasets/gauravduttakiit/resume-dataset)**

2. Download `UpdatedResumeDataSet.csv` and place it in this `data/` folder:
   ```
   data/UpdatedResumeDataSet.csv
   ```

3. Then run the preprocessing and training scripts:
   ```bash
   python notebooks/01_preprocessing.py   # → data/processed_resumes.csv
   python notebooks/02_model.py            # → models/resume_model.pkl
   python notebooks/03_bert_model.py       # → models/bert_resume_model.pkl  (~90 MB download on first run)
   ```

## Notes

- `bert_embeddings.npy` (generated by `03_bert_model.py`) is also gitignored — it is ~1.4 MB and can be regenerated at any time.
- The pre-built `.pkl` files in `models/` were trained with the **Pipeline** approach so the TF-IDF vectoriser is fitted only on the training split (no data leakage). If you retrain, the same isolation is enforced by the script.
